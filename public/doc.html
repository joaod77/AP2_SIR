Documentação do Sistema de Gestão de Estudantes

Como usar o sistema

Este sistema utiliza a tecnologia LowDB para persistir os dados localmente em um arquivo `db.json`. As operações CRUD permitem que você adicione, edite, visualize e exclua informações dos estudantes.

Endpoints da API:

  • GET /students - Retorna todos os estudantes cadastrados.
  • GET /students/:id - Retorna um estudante específico por ID.
  • POST /students - Cria um novo estudante.
  • PUT /students/:id - Atualiza um estudante específico por ID.
  • DELETE /students/:id - Deleta um estudante específico por ID.
