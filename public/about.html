Sobre o Sistema de Gestão de Estudantes

O que é este sistema?

Este é um sistema de gestão de estudantes que permite o armazenamento de informações sobre alunos, incluindo nome, curso e ano de estudo. Ele fornece operações de CRUD (Criar, Ler, Atualizar, Deletar) para gerenciar esses dados.
